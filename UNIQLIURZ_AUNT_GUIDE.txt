=== TASK 3: FIX ROOT DOMAIN ===

Purpose: So typing "uniqliurz.com" (without www) also shows the website

Step-by-step for aunt:

1. Go to: https://console.cloud.google.com
2. At the top, in the search bar, type: [PASTE THIS] Cloud DNS
3. Click on "Cloud DNS" in the results
4. You'll see a list - click on "uniqliurz.com" (it might show as a zone name)
5. You'll see a table of DNS records. Look for a row where:
   - Type column says "A"
   - DNS Name column is blank or shows "@"
6. If you see that row with "23.227.38.65" in the Data column:
   - Click the pencil/edit icon on the right
   - Change "23.227.38.65" to "76.76.21.21"
   - Click "Save"
7. If you DON'T see that row:
   - Click "Add Record Set" at the top
   - Fill in:
     - DNS Name: leave it BLANK
     - Type: select "A"
     - TTL: type "300"
     - IPv4 Address: type "76.76.21.21"
   - Click "Create"
8. Wait 10-30 minutes, then open a new tab and visit: https://uniqliurz.com
9. If it shows the website, it worked!
